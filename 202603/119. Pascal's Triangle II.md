# 119. Pascal's Triangle II

* Time: O(rowIndex^2) (1 + 2 + 3 + ... + n)
* Space: O(rowIndex)


```python3
class Solution:
    def getRow(self, rowIndex: int) -> List[int]:
        row = [1]
        for i in range(1, rowIndex + 1):
            tmp = row
            row = [1]
            for j in range(1, i):
                row.append(tmp[j - 1] + tmp[j])
            row.append(1)

        return row

```
